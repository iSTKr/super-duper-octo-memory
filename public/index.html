This site is deployed using GitHub Actions pipeline. It launches AWS EC2 instance via Terraform main.tf module, then creates and deploys Docker container inside EC2 instance via Dockerfile. The github repository with this site code checkouts inside container and launches the HTTP server using apache2. Also, the GitHub Actions pipeline updates the site code after every push to the repository.
